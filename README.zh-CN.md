### 数字计算机的发展

> **图灵机**

1. 1946 ~ 1957： 电子管，**机器语言**编程，无操作系统
2. 1958 ~ 1964： 晶体管， 单道批处理系统
3. 1965 ~ 1971：集成电路， 多道批处理系统和分时系统
4. 1972 ~ : 超大规模集成电路

> 摩尔定律: 集成电路上可容纳的晶体管数量大约每两年翻一番
